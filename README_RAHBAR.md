# Nour English — бот «Рахбар»

Полнофункциональный бот в духе @CrossEnglish_bot, адаптированный под
Nour English: наставник «Рахбар» определяет регион, показывает меню
из 4 опций, проводит тест на 30 вопросов (CEFR A1–C1) и направляет
на канал.

## Что умеет бот

1. **Приветствие + выбор региона** (для часового пояса)
   — Россия / Беларусь / Казахстан
   — Европа
   — Остальной мир

2. **Главное меню** из 4 пунктов:
   — 📜 Узнать свой уровень (тест 30 вопросов)
   — 💰 Цены и программы
   — 🤝 Записаться на пробный
   — ✍️ Написать преподавателю (прямая ссылка на тебя)

3. **Тест на 30 вопросов**, сложность растёт от лёгкого к сложному.
   В конце — уровень по шкале CEFR (A1–C1) с описанием.

4. **После теста** — выбор направления (ЕГЭ, IELTS, переезд,
   разговорный, с нуля) → переход на канал Nour English.

## Шаг 1 — создать бота через BotFather

1. Открой `@BotFather` в Telegram
2. Напиши `/newbot`
3. Дай имя, например `Rahbar — Nour English`
4. Дай username, например `NourEnglishRahbar_bot`
5. Скопируй токен — выглядит так: `123456789:ABC-DEF1234ghIkl...`

## Шаг 2 — вставить свои данные

Открой файл `bot_rahbar.py`, в самом начале найди:

```python
BOT_TOKEN = os.environ.get("BOT_TOKEN", "ВСТАВЬ_СЮДА_СВОЙ_ТОКЕН_ОТ_BOTFATHER")
CHANNEL_LINK = "https://t.me/NourEnglish"
TEACHER_USERNAME = "https://t.me/Nazir"
```

Замени на:
- свой токен от BotFather
- финальную ссылку на канал
- свой личный username для связи (кнопка «Написать преподавателю»)

## Шаг 3 — заполнить цены и программы

В функции `show_prices()` есть раздел с примером цен — замени его
на свои реальные программы и стоимость.

## Шаг 4 — запустить локально (для проверки)

```bash
pip install -r requirements.txt
python bot_rahbar.py
```

Если видишь «Бот Рахбар запущен» — открой бота в Telegram и напиши `/start`.

## Шаг 5 — разместить на бесплатном хостинге (чтобы бот работал 24/7)

### Вариант А — Railway.app (проще всего)
1. Зарегистрируйся на railway.app через GitHub
2. Залей этот код в свой репозиторий на GitHub
3. В Railway: New Project → Deploy from GitHub repo
4. В Variables добавь `BOT_TOKEN` = твой токен
5. Railway сам запустит `python bot_rahbar.py`

### Вариант Б — PythonAnywhere
1. Зарегистрируйся на pythonanywhere.com (бесплатный план)
2. Загрузи файл через Files
3. Bash-консоль → `pip install python-telegram-bot`
4. Запусти через Always-on task (на платном плане) либо Scheduled task

### Вариант В — собственный VPS
Для роста аудитории — недорогой VPS (от 200–300 руб/мес),
бот запускается через `systemd` или `screen` в фоне постоянно.

## Как менять вопросы теста

Все вопросы — в списке `QUESTIONS` в файле `bot_rahbar.py`:

```python
{"q": "Текст вопроса", "options": ["в1", "в2", "в3", "в4"], "correct": 0, "level": 1},
```

`correct` — индекс правильного варианта (с нуля).
`level` — сложность вопроса (1 лёгкий → 5 сложный), влияет на итоговый балл.

## Как менять границы уровней CEFR

Таблица `LEVEL_TABLE` определяет, сколько баллов соответствует
какому уровню. Можно настроить под себя.
